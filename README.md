This project is a **Frontend clone of the IRCTC homepage**, built during my **internship**.

The main objective of this project was to **design and replicate the UI of the IRCTC website** using HTML, CSS, and JavaScript, focusing on layout, styling, and user interface structure.

## ✨ Features

- 🚆 IRCTC-style homepage UI design  
- 🧭 Navigation bar with multiple sections  
- 📅 Train booking form (From, To, Date, Class, Category)  
- 🔄 Interactive UI elements  
- 🕒 Live date & time display  
- 🎨 Responsive and clean layout  
- 📦 Multiple service sections (Flights, Hotels, Bus, Holidays, etc.)  
- 📱 Social media and footer section  

---

## 📚 Concepts Used

- HTML semantic structure  
- CSS Flexbox & Grid  
- Responsive layout design  
- DOM manipulation using JavaScript  
- Event handling  
- UI/UX design principles  

---

## ⚙️ How It Works

- The UI is built using structured HTML elements  
- Styling is done using CSS for layout and design  
- JavaScript is used to display real-time date and time
